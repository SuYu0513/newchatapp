# PWAインストールガイド

## 📱 アプリのようにインストールして使う方法

このチャットアプリは**PWA（Progressive Web App）**対応なので、スマホやPCにインストールして、ネイティブアプリのように使えます！

---

## 💻 Windowsでのインストール方法

### Chrome / Edge

1. **サーバーを起動**
   ```bash
   ./mvnw spring-boot:run
   ```

2. **ブラウザでアクセス**
   - `http://localhost:9090` を開く

3. **インストール**
   - アドレスバー右側の **「インストール」アイコン** をクリック
   - または **メニュー > アプリをインストール** を選択

4. **完了！**
   - デスクトップにショートカットが作成されます
   - スタートメニューにも追加されます

### インストール後の使い方

✅ **デスクトップアイコンをダブルクリック**するだけで起動
✅ **独立したウィンドウ**で開く（ブラウザのUIなし）
✅ **スタートメニュー**から検索して起動
✅ **タスクバーにピン留め**も可能

---

## 📱 スマホ（Android/iOS）でのインストール方法

### Android（Chrome）

1. サーバーを起動した状態で、スマホのブラウザから `http://<PCのIPアドレス>:9090` にアクセス
2. 画面下部に「ホーム画面に追加」が表示される
3. 「インストール」をタップ
4. ホーム画面にアプリアイコンが追加される

### iOS（Safari）

1. サーバーを起動した状態で、Safariからアクセス
2. 共有ボタン（□に↑）をタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ

---

## ⚠️ 重要な注意点

### サーバーは起動が必要

PWAとしてインストールしても、**バックエンドサーバー（Spring Boot）は別途起動する必要があります**。

**理由：**
- チャット機能はリアルタイム通信（WebSocket）が必要
- データベースへのアクセスが必要
- ユーザー認証が必要

### 解決策：2つの方法

#### 方法1: バッチファイルで自動起動（推奨）

`start-server.bat` を作成：
```batch
@echo off
cd /d "C:\Users\y_sudo_462\Documents\VSコード\自主課題\チャットアプリ\chatapp"
start cmd /k mvnw spring-boot:run
```

Windowsスタートアップに登録すれば、PC起動時に自動で起動します。

#### 方法2: Windowsサービス化

完全にバックグラウンドで動作させたい場合は、Windowsサービスとして登録できます。
（詳細は `WINDOWS-SERVICE.md` を参照）

## 🚀 PWAの利点

✨ **すぐ起動** - アイコンをクリックするだけ
✨ **独立ウィンドウ** - ブラウザのUIがない
✨ **オフラインキャッシュ** - 静的リソースが高速

## 📋 インストール確認

インストールが成功すると：

- ✅ デスクトップにアイコンが表示
- ✅ スタートメニューに追加
- ✅ 独立したウィンドウで起動
- ✅ タスクバーに別のアイコンで表示
- ✅ アンインストールも簡単（設定 > アプリから）

## 🔧 トラブルシューティング

### インストールボタンが表示されない

- HTTPSまたはlocalhostで動作している必要があります
- `manifest.json`が正しく読み込まれているか確認
- Service Workerが登録されているか確認（DevTools > Application > Service Workers）

### アプリが起動しない

1. **サーバーが起動しているか確認**
   ```bash
   ./mvnw spring-boot:run
   ```

2. **ブラウザで直接アクセスできるか確認**
   - `http://localhost:9090`

3. **キャッシュをクリア**
   - DevTools > Application > Clear storage

## 💡 さらに便利に使うには

### 自動起動スクリプトを作成

`quick-start.bat`:
```batch
@echo off
echo 🚀 チャットアプリを起動中...
cd /d "%~dp0"
start "" "http://localhost:9090"
start cmd /k mvnw spring-boot:run
```

ダブルクリックするだけで：
1. サーバーが起動
2. ブラウザが自動で開く
